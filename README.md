### AVAILABLE ON :

* Termux

### REQUIREMENTS :
* Zsh shell
* Storage 10 MB
* Pkg - Git ; Wget ; Lolcat ; Zsh ; etc

### FEATURES :
* [+] 10+ Banners ( I will add more soon ) !
* [+] Pure Zsh Theme !
* [+] Auto Suggestion | Highlight !
* [+] Fully Automatic !
* [+] Easy for Beginners !

## INSTALLATION [Termux] :

* `apt-get update -y`
* `apt-get upgrade -y`
* `git clone https://github.com/h4ck3r0/Termux-banner`
* `cd $HOME`
* `ls`
* `cd Termux-banner`
* `chmod +x *`
* `bash setup.sh`
* `bash banner.sh`
```
[+]-- Open new session in termux and Check banner 
```
